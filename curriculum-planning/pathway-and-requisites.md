## 🛣️ The "Student Success" Map (Recommended Sequence)
*This sequence optimizes the flow of "Advisories" and manages cognitive workload across the certificate.*

### **Semester 1: The Gateway (Sequential 9-Week Blocks)**
* **CIT 82:** Intro to AI & Web (Weeks 1-9)
* **CIT 85:** Applied HTML/CSS (Weeks 10-18)
* *Note: Establishes the "Mechanical Shop" workflow (Git/VS Code) early.*

### **Semester 2: The Logic Pillar (Full 18-Week Term)**
* **CIT 93:** JavaScript.
* *Pedagogical Focus: Isolated focus on programming logic to prevent cognitive overload.*

### **Semester 3: Data & Integration (Full 18-Week Term)**
* **CIT 28:** SQL (Data Modeling).
* **CIT 84:** AI Web Workflows (Capstone Frameworks Course).
* *Pedagogical Focus: High-level synthesis. CIT 93 provides the coding foundation, CIT 28 provides the data layer, and CIT 84 ties them together using component-driven frameworks and AI-accelerated workflows.*

---

## 🧠 Workgroup Rationale

1. **The Core Entry Tier:** By keeping **CIT 48A / 15 / 82** as the hard prerequisite for the technical core, we ensure every student has mastered foundational OS and file-system navigation before entering lab-intensive courses.
2. **The Capstone Gatekeeper:** Moving the workflows capstone to **CIT 84** leverages its existing, state-approved prerequisite of **CIT 93 (JavaScript)**. This guarantees that students are not attempting high-level framework architectures or prompt orchestration before mastering baseline functional programming logic.
3. **Context-Specific Database Advisories:** While CIT 28 is a first-term course for Data Analytics, positioning it in Semester 3 for Web Developer students allows it to run concurrently with CIT 84. This allows students to build real-time client-server interfaces that communicate directly with their database labs.
4. **Cognitive Load Management:** The sequence is designed to isolate **CIT 93 (JavaScript)** in Semester 2. By separating it from database modeling and frameworks, we maximize pass rates in our most challenging foundational logic course before accelerating student capabilities in the third term.
