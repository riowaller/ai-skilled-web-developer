## 1. The Core Pathway Sequence Lock-In (4-Semester Fixed Cohort Model)
*Objective: Solidify the exact unit loads and course expectations for the newly streamlined, elective-free track.*

*   **Semester 1 (The Foundational Duo):** CIT 82 (Intro to AI and Web Development) + CIT 85 (HTML/CSS). Establishes environmental baseline literacy, core prompting frameworks, and web page layout fundamentals.
*   **Semester 2 (Data & Logic Foundations):** CIT 84 (Re-imagined as Coding Concepts) + CIT 28 (SQL). Students master language-agnostic logic, algorithmic flow, and structured database queries simultaneously *before* handling complex programming language syntax rules.
*   **Semester 3 (The Logic Pillar):** CIT 93 (JavaScript). Runs in complete syntax isolation to prevent cross-language cognitive overload.
*   **Semester 4 (The Advanced Capstone):** CIT 94 (Agentic AI). Replaces previous capstone configurations. Focuses on full-stack application architecture, autonomous orchestration, state management, and critical code security auditing.

## 🧠 Core Pedagogy: The "Mechanical Shop"
We are training students to navigate an **AI-Augmented Career Ecosystem**.
1.  **Foundational Mastery:** Manual authoring of code to understand "under-the-hood" mechanics.
2.  **AI Orchestration:** Using LLMs for debugging, explanation, and efficiency—not primary generation.
3.  **Verifiable Competency:** Performance-based mastery via **Verbal Code Reviews** and **Technical Interviews**.
