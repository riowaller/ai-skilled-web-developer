## Lecture & Lab Content Revision



| **Topic Area** | **Proposed Sub-topics** | **Rationale for Change** |
| :--- | :--- | :--- |
| **Database Design** | Entity Relationship Diagrams (ERD), Data Types, Primary/Foreign Keys. | The "Planning" phase of database creation. |
| **Data Organization** | Reducing data redundancy and organizing tables for efficiency. **Using AI to audit schema logic.** | Replaces formal Normalization theory with practical design skills. |
| **Basic SQL (DML)** | SELECT, INSERT, UPDATE, DELETE. **Using AI for syntax and error explanation.** | The "Mechanical Shop" of data manipulation and writing. |
